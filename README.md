UPI Growth & User Behaviour Analysis Dashboard
Overview
This Power BI dashboard analyzes UPI (Unified Payments Interface) transaction data to uncover transaction trends, user behavior, app performance, banking insights, and regional transaction patterns across India.
The project provides interactive visualizations that help understand how users engage with UPI platforms, identify transaction success rates, monitor growth trends, and evaluate banking and state-level performance.

Objectives
* Analyze UPI transaction growth across India. 
* Study user transaction behavior and usage patterns. 
* Evaluate transaction success and failure rates. 
* Compare performance of different UPI applications. 
* Identify high-performing states and banks. 
* Discover transaction trends by time, age group, and device preference. 

Tools & Technologies

Dashboard Pages
1. UPI Growth in India
This page provides an overall performance summary of UPI transactions.
Key Metrics
* Total Transactions: 20K 
* Total Transaction Amount: 50.03M 
* Average Transaction Amount: 2.50K 
* Success Rate: 89.70% 
* Failure Rate: 7.39% 
Visualizations
* Transaction distribution by UPI App 
* Transaction status analysis 
* State-wise transaction distribution 
* Sender bank performance summary 
* Receiver bank transaction summary 
Insights
* PhonePe contributes the largest share of transactions. 
* Success transactions significantly exceed failed transactions. 
* Maharashtra and Karnataka are among the leading states in transaction volume. 
* SBI handles the highest transaction amount among sender banks. 

2. UPI User Behaviour & Usage Pattern
This page focuses on user engagement and transaction behavior.
Visualizations
* Hourly Transaction Trend 
* Monthly Transaction Trend 
* UPI Usage by Age Group 
* Device Preference Analysis 
* Average Transaction Amount by Day 
Insights
* Users aged 26–35 contribute the highest transaction volume. 
* Android is the dominant platform for UPI transactions. 
* Transaction activity varies throughout the day with identifiable peak hours. 
* Average transaction amount is highest on weekends. 
* Monthly transaction volume remains relatively stable with slight seasonal fluctuations. 

Business Insights
User Behavior
* Young adults (18–35 years) form the majority of UPI users. 
* Mobile devices, particularly Android, dominate digital payment adoption. 
Transaction Performance
* High success rate indicates strong reliability of UPI services. 
* Failed transactions represent a small percentage of total activity. 
Banking Analysis
* Major banks such as SBI, ICICI, HDFC, Axis, and PNB contribute significantly to transaction processing. 
* Differences in transaction volume help identify leading banking partners. 
Regional Analysis
* Certain states contribute a larger share of UPI transactions, indicating stronger digital payment adoption.
